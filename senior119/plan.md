# PROJECT: 시니어 119 (Senior 119) 개발 명세서

## 1. 프로젝트 개요
- **서비스명:** 시니어 119 (Senior 119)
- **한줄 소개:** 65세 이상 어르신을 위한 퇴직금, 실업급여, 부당해고 법률 자가진단 "법률 응급실" 서비스
- **핵심 타겟:** 스마트폰 사용이 미숙하고 노안이 있는 65세 이상 근로자 & 그들의 자녀
- **플랫폼:** 모바일 웹 (Mobile Web) - 별도 설치 없이 링크로 접속
- **목표:** 복잡한 노동법을 병원 진단처럼 쉽게(O/X) 풀어서 해결책(처방전)을 제시하고, 카카오톡으로 쉽게 공유하게 함.

## 2. 디자인 시스템 & UI/UX 가이드라인 (최우선 순위)
**"무조건 크고, 직관적이고, 움직여야 한다."**

### 2.1. 타이포그래피 (Typography)
- **폰트:** `Pretendard` 또는 `Noto Sans KR` (가독성 최우선)
- **사이즈 정책:**
  - 제목(Header): 최소 `28px` (Bold)
  - 본문(Body): 최소 `20px` (Medium) - 절대 16px 이하 사용 금지
  - 버튼 텍스트: `24px` 이상

### 2.2. 컬러 팔레트 (High Contrast)
- **Primary (브랜드):** `Royal Blue (#0056B3)` - 신뢰, 경찰/구급차 느낌
- **Action (응급/해고):** `Emergency Red (#FF4D4D)` - 긴급함, 위험
- **Warning (임금/퇴직금):** `Alert Orange (#FFA500)` - 주의, 돈
- **Safe (실업급여/가능):** `Safety Green (#28A745)` - 안전, 통과
- **Background:** `White (#FFFFFF)` or 아주 연한 회색 (눈의 피로도 감소)

### 2.3. 인터랙션 & 효과 (Animation & SFX)
- **애니메이션:** `Framer Motion` 활용. 화면 전환 시 부드럽게 넘어가기보다, 버튼을 눌렀을 때 **"확대되었다가 줄어드는(Scale Up/Down)"** 확실한 피드백 제공.
- **사운드 (옵션):** 버튼 클릭 시 "띵동" 같은 긍정적 청각 피드백 (Mute 기능 포함).
- **진동:** 모바일 접속 시 버튼 클릭 시 햅틱 피드백(Vibration) 적용.

## 3. 핵심 기능 및 페이지 구성 (Sitemap)

### 3.1. 메인 화면 (Home) - "무엇이 불편하신가요?"
- **헤더:** [시니어 119] 로고 (구급차 + 돋보기 심볼)
- **메인 카피:** "어르신! 떼인 돈, 억울한 일 있으세요? 제가 봐드릴게요."
- **3대 증상 버튼 (화면의 80% 차지):**
  1. 🔴 **[갑자기 해고 당했어요!]** (부당해고 진단)
  2. 🟠 **[퇴직금을 안 줘요!]** (퇴직금/임금체불 진단)
  3. 🟢 **[실업급여 받을 수 있나요?]** (수급자격 진단)
- **하단 푸터:** "자녀분에게 이 앱 보내주기" (카카오톡 공유 버튼)

### 3.2. 진단 화면 (Diagnosis) - "증상 체크"
- **진행 방식:** 한 화면에 **질문 딱 1개**만 표시 (Step by Step).
- **UI 구성:**
  - 상단: 진행률 바 (지렁이처럼 채워지는 굵은 바)
  - 중앙: 질문 텍스트 (예: "65세 생일이 지나고 취업하셨나요?")
  - 하단: 초대형 **[O / X]** 버튼. (O는 파랑/초록, X는 빨강)
- **TTS 기능:** 질문 옆 스피커 아이콘 클릭 시 음성으로 질문 읽어주기.

### 3.3. 결과 화면 (Prescription) - "처방전"
- **컨셉:** 약봉투 또는 의사 소견서 디자인.
- **구성:**
  - **진단 결과:** "받을 수 있습니다!" (큰 아이콘 + 텍스트)
  - **상세 소견:** "사장님이 OOO라고 했다면 불법입니다." (쉬운 설명)
  - **행동 요령:** "지금 당장 노동청(1350)에 전화하세요."
  - **Action Button:**
    - [아들/딸에게 결과 보내기] (카카오톡 공유 - 결과 내용 포함)
    - [가까운 노무사 찾기] (추후 확장)

## 4. 기술 스택 (Tech Stack)
- **Frontend:** React (Vite), JavaScript
- **Styling:** Tailwind CSS (빠른 스타일링), Styled-components
- **Animation:** Framer Motion (큰 이펙트 구현용)
- **Logic:** Rule-based Logic (별도 백엔드 없이 클라이언트 사이드에서 `if-else` 로직 처리)
- **Deployment:** Vercel (빠른 배포 및 수정)
- **Share:** KakaoTalk JavaScript SDK (카카오톡 링크 공유 기능 필수)

## 5. 카카오톡 공유 최적화 (SEO & OG Tag)
- **목적:** 자녀들이 링크를 봤을 때 "스팸"이 아니라 "부모님 필수 앱"처럼 보여야 함.
- **Open Graph 설정:**
  - `og:title`: "시니어 119 - 어르신 법률 응급실"
  - `og:description`: "부당해고, 퇴직금, 실업급여... 우리 부모님은 해당될까요? 1분 만에 확인해보세요."
  - `og:image`: 구급차가 달려가는 3D 일러스트 또는 돋보기 이미지 (신뢰감 주는 파란색 톤)

## 6. 개발 로드맵 (Phase 1)
1.  **Project Setup:** React + Tailwind + Framer Motion 설치.
2.  **Logic Definition:** 실업급여/퇴직금 관련 O/X 로직 JSON 데이터화.
3.  **UI Implementation:**
    - 메인 화면 (3버튼) 구현.
    - O/X 퀴즈 컴포넌트 구현 (애니메이션 포함).
    - 결과 페이지 구현.
4.  **Kakao Share Integration:** 공유하기 버튼 연동.
5.  **Test:** 모바일 환경(특히 폰트 크기) 테스트.
